## Project Overview
The Site Survey Tool is a software application designed to assist Internet Service Providers (ISPs)
in planning and managing network installations across large areas such as apartment complexes,
office buildings, campuses, and public spaces.

The tool enables engineers to collect, manage, and analyze critical site-related information
before actual network deployment, helping reduce installation cost, time, and errors.

---

## Key Features
- Site and property management
- Building, floor, and space hierarchy
- Floor plan upload (Image/PDF)
- Equipment placement planning
- Checklist-based site survey
- Data validation and reporting support

---

## Tech Stack
- **Frontend:** React.js + Tailwind CSS  
- **Backend:** Spring Boot (Java)  
- **Database:** MySQL  
- **Authentication:** JWT (Access & Refresh Tokens)
